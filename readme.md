# elva-labs homebrew tap

## Packages

- [awsesh](https://github.com/elva-labs/awsesh)

## Installation

To install the tap, run the following command:

```bash
brew tap elva-labs/elva
```

Then install the desired package, ex:

```bash
brew install awsesh
```
